## Лабораторная работа №4: Интерфейсы и абстрактные классы

*Интерфейс* — это способ описания поведения объектов через набор методов, которые должны быть реализованы в классах. В Python интерфейсы реализуются с помощью абстрактных базовых классов (ABC).                

*Абстрактный класс* — класс, который не может быть создан напрямую и содержит абстрактные методы, обязательные для реализации в дочерних классах.                

В данной работе используются интерфейсы Printable и Comparable.                

### Интерфейс Printable
Определяет возможность представления объекта в виде строки.                

Позволяет всем классам реализовать единый способ вывода информации, но с разной реализацией.                

### Интерфейс Comparable
Определяет возможность сравнения объектов между собой.                

Позволяет реализовать сортировку и сравнение без использования условных конструкций.                

### Полиморфизм через интерфейсы
Объекты разных классов могут реализовывать одинаковые методы интерфейсов, но с разной логикой.                

Это позволяет работать с ними одинаково, не зная конкретного типа объекта.                

### isinstance()
Используется для проверки, реализует ли объект определённый интерфейс.                

---

## Реализация в работе

Интерфейсы реализованы в следующих классах:

- CityBus                
- TouristBus                
- ElectricBus                

Все классы наследуются от базового класса Bus и реализуют интерфейсы Printable и Comparable.                

---

### Реализация Printable

Каждый класс по-своему реализует вывод информации:

- CityBus — выводит маршрут и количество пассажиров                
- TouristBus — дополнительно выводит информацию о гиде                
- ElectricBus — выводит уровень экологичности                

---

### Реализация Comparable

Сравнение объектов реализовано по единому критерию — доходу.                

Это обеспечивает:
- корректную сортировку объектов                
- единый интерфейс сравнения                
- отсутствие ошибок при сравнении разных типов
