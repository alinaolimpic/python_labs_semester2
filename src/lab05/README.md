## Лабораторная работа №5: Функции как аргументы. Стратегии и делегаты

В данной лабораторной работе изучается использование функций как аргументов, а также применение функционального подхода в сочетании с объектно-ориентированным программированием.

Рассматриваются:
- функции высшего порядка
- lambda-выражения
- фабрики функций
- паттерн «Стратегия»
- callable-объекты

---

## Реализованные функции и стратегии

### Стратегии сортировки

Реализованы стратегии сортировки объектов коллекции:
- по вместимости
- по скорости
- по имени водителя
- по нескольким параметрам одновременно

Сортировка выполняется за счёт передачи функции как аргумента.
---

### Фильтрация

Реализованы функции фильтрации:
- по вместимости
- по скорости
- по типу объекта

Фильтрация выполняется через передачу предикатов.

---

### Функции высшего порядка

В работе используются:
- map — для преобразования коллекции
- filter — для отбора элементов
- lambda-выражения для упрощённых операций

---

### Фабрика функций

Реализована функция, создающая фильтр с параметром.  
Это позволяет динамически задавать условия фильтрации.

---

### Паттерн «Стратегия»

Реализован через callable-объекты.

Позволяет:
- изменять поведение без изменения кода коллекции
- подставлять разные алгоритмы обработки

---

### Расширение коллекции

Коллекция расширена и поддерживает:
- сортировку через передачу функции
- фильтрацию через предикаты
- применение функций к каждому элементу
- цепочку операций
