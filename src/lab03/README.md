## 2. Описание иерархии

Базовый класс:
- BaseBus (расширяет Bus)

Дочерние классы:
- CityBus
  - количество остановок
  - тариф
  - расчет дохода

- IntercityBus
  - дистанция
  - уровень комфорта
  - расчет времени поездки
