# Лабораторная работа №3: Наследование классов

*Наследование* — это способ построения новой программной сущности на основе существующей, с возможностью добавления новых полей и методов, а также изменения унаследованного поведения.                  

*Базовый класс* (родитель) — класс, от которого наследуют. Содержит общие характеристики для группы объектов. В данной работе это Bus.                   

*Производный класс* — (дочерний, подкласс) — класс, который наследует от базового. Добавляет специфичные атрибуты и методы. Например, CityBus, TouristBus, ElectricBus.                  

### Переопределение методов
Производный класс может переопределить (заменить) метод базового класса, чтобы изменить его поведение. Для этого достаточно объявить метод с тем же именем в дочернем классе. Вызов родительской реализации можно осуществить через super().                  

### Полиморфизм
Способность объектов разных классов отвечать на один и тот же метод по-своему.                  
Пример: `process()` или `calculate_income()` для разных автобусов работает по-разному.                   

### super()
Вызов метода родительского класса. Используется в конструкторе `__init__` для инициализации унаследованных атрибутов.                   

### isinstance()
isinstance(object, classinfo) проверяет, является ли объект экземпляром указанного класса или его потомка.                   

### Атрибуты класса и атрибуты экземпляра
*Атрибуты класса* — общие для всех объектов.                    
*Атрибуты экземпляра* — уникальные для каждого объекта (создаются в `__init__`).                    

## Иерархия классов в работе

### Базовый класс Bus
Содержит общие атрибуты:                    
- маршрут, вместимость, скорость, водитель, пассажиры, статус.                 

Методы:                    
- board_passengers(), start_route(), calculate_income(), __str__()                    

### класс CityBus
Новые атрибуты:                 
- __stops_count — количество остановок                    
- __ticket_price — цена билета                  

**Новый метод:** calculate_income() — рассчитывает доход от пассажиров.                 

**Переопределённый** `__str__` — добавляет информацию об остановках.                   

### класс TouristBus
Новые атрибуты:                 
- __guide_name — гид                    
- __tour_price — цена тура                   

**Новый метод:** calculate_income() — доход от туров.                 

**Переопределённый** `__str__` — добавляет информацию о гиде.                 

### класс ElectricBus
Новые атрибуты:                 
- __battery_capacity — ёмкость батареи                  
- __eco_level — уровень экологичности                    

**Новый метод:** calculate_income() — с учётом экологического бонуса.                   

**Переопределённый** `__str__` — добавляет экологические параметры.                  

### Методы классов

`CityBus` - calculate_income()                  
`TouristBus` - calculate_income()                  
`ElectricBus` - calculate_income()                   

### Переопределённые методы
Отличия от базового класса метод calculate_income()                 
`CityBus` учитывает количество пассажиров                 
`TouristBus` учитывает стоимость тура                 
`ElectricBus` учитывает экологический бонус
